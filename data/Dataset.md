# Dataset

The dataset used in this project exceeds GitHub's 25MB file size 
limit and could not be uploaded directly to this repository.

**Download the dataset here:**
[Home Credit Default Risk — application_train.csv](https://www.kaggle.com/competitions/home-credit-default-risk/data?select=application_train.csv)

Once loaded into MySQL, refer to the scripts folder for 
`home_credit_data_cleaning.sql` and `home_credit_eda.sql` — each 
containing the full query logic, transformations, and inline 
documentation for every analytical decision made in this project.
